## 使用說明

1. **在 Word 模板中添加變數標記**：
   - 打開 `ENT_Clinic_Record_Design_Portrait_Fixed.docx`
   - 找到對應的位置（例如：病歷號碼欄位）
   - 在該位置輸入或替換為：`{{ patient_id }}`
   - 重複此步驟，為每個位置添加對應的變數標記

2. **保存模板**：
   - 保存修改後的 Word 文件
   - 將文件複製到 `templates/` 目錄（如果還沒有）

3. **告訴我每個位置對應的數據**：
   - 如果模板中的變數名稱與上表不同，請告訴我實際使用的變數名稱
   - 如果有其他需要填充的位置，請告訴我

## 注意事項

- 變數名稱必須使用 `{{ variable_name }}` 格式（Jinja2 語法）
- 變數名稱區分大小寫
- 如果某個位置不需要填充，可以留空或使用 `{{ "" }}`
